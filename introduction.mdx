# QuickJS-ng - A Mighty JavaScript Engine

QuickJS is a small and embeddable JavaScript engine that aims to support the latest [ECMAScript specification](https://tc39.es/ecma262/). This project is a fork of the original QuickJS project by Fabrice Bellard and Charlie Gordon, revitalized with the intent of reigniting its development.

## Quick Example

Here's a simple example of embedding QuickJS in a C application:

```c
#include "quickjs.h"

int main() {
    JSRuntime *rt = JS_NewRuntime();
    JSContext *ctx = JS_NewContext(rt);
    
    const char *code = "console.log('Hello from QuickJS!')";
    JSValue result = JS_Eval(ctx, code, strlen(code), "<input>", JS_EVAL_TYPE_GLOBAL);
    
    JS_FreeValue(ctx, result);
    JS_FreeContext(ctx);
    JS_FreeRuntime(rt);
    
    return 0;
}
```
